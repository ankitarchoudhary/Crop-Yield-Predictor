# 🌾 Fasal Upaj Predictor
### AI-Powered Hyperlocal Crop Yield Predictor for Indian Farmers

**Developer:** Ankita Choudhary | AI & Data Science — IIT Guwahati

## 🚀 Features
- 📍 Village → Tehsil → District → State granularity
- 📏 State-wise local land units (Killa, Katha, Cent, Guntha...)
- 🗣️ 22 Indian languages support
- 🤖 AI Chat Assistant (KrishiBot)
- 📊 Real government data (data.gov.in)
- 🌾 25+ crops supported

## 🛠️ Tech Stack
- Streamlit, Python, Scikit-learn, Groq AI, Plotly
